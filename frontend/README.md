# PeerNet Frontend

React + Vite frontend for PeerNet.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start Development Server:
   ```bash
   npm run dev
   ```

## Features

- **Authentication**: Login and Register
- **Learning**: Browse available sessions
- **Teaching**: Create new sessions
- **Profile**: View user stats and skills

## Tech Stack

- React 18
- Vite
- Tailwind CSS (via CDN)
- React Router DOM
- Axios
